== Installation ==

= Creating a comparison table =

1. Create a free account at [Compare Ninja](http://compareninja.com).
2. Create a new comparison table by clicking the "Start Now" button on Compare Ninja's homepage.
3. Customize your table. Add rows and columns, and choose skin.
4. Click the "Generate Table" button.
5. In the "Table Publish" page look for the "Table ID" section. Copy it, and write it somewhere (we will use it later).

= Attach a comparison table to a Wordpress post =

1. Create a new post and look for the "Comparison Table" button in your editor.
2. A popup will appear, enter the relevant tableID (the same one you copied from Compare Ninja).
3. If you can't find the "Comparison Table" button, you may also write [compareninja tableid="YOUR_TABLE_ID"] in the editor, and replace "YOUR_TABLE_ID" with the relevant table ID.

Arguments provides by the shortcode:

* 'tableid' Your table ID from Compare Ninja
