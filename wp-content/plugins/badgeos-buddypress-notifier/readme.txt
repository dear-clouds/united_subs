== Description ==

A simple WordPress plugin to enable BuddyPress on-site notifications when a user reaches an achievement with BadgeOS-Community.

= Usage =

Plugin can be simply activated and used. There are no configuration options, you should see on site notifications
apear whenever a user is awarded a new badge. This can be stimulated for testing purposes, by going to:
`Users -> [USERNAME] -> Award Badge`

= Requirements =

* BuddyPress
* BadgeOS
* BadgeOS Community Add-on

== Installation ==

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Is this pluging provide AJAX notifcations? =

No. BuddyDev have already done a great job of providing this functionality and I take extra care to test that BadgeOS BuddyPress Notifier is
compatable with BuddyDev's ajax plugin.

[BuddyDev - BuddyPress Live Notification](http://buddydev.com/plugins/buddypress-live-notification/)
