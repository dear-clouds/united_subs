A small plugin without settings will allow you to customize your answers on the forum in special style.

== Description ==

The plugin allows you to isolate the replies of the administration or moderators of a particular color, plugin adding a checkbox to the form of the answer. You can upload your styles in style.css and change the language in  .po files. 

If you have the ability to translate to your language please contact me on VK or daniluk4000@outlook.com

== Frequently Asked Questions ==

= How do I activate a new style? =

Create the reply, go to the edit page (front end) and activate the checkbox.

= How can I change the style? =

Simply edit the styles in style.css. Each style must contain .bbp-admin-reply!
