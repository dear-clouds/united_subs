Adds tab in the Buddypress groups for external blog RSS feeds posts of group activity

== Description ==
Adds tab in group for external blog RSS feeds posts of group activity. Requires External Group Blogs plugin (http://wordpress.org/extend/plugins/external-group-blogs/) to by installed.


== Installation ==
1. Upload plugin folder `/external-group-rss-tab/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
